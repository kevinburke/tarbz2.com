If your file looks like:

example.tar.bz2
→

Use the following command:

tar -xvjf example.tar.bz2

If your file looks like:

example.tar.gz
→

Use the following command:

tar -zxvf example.tar.gz

If your file looks like:

example.zip
→

Use the following command:

unzip example.zip
